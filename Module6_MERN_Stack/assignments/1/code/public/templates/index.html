Module 6 - Assignment 1

Building a Full-Stack Node.js Web Server

Site Description

This site demonstrates the ability to create a web server using Node.js. The purpose is to fulfill the needs of the first assignment in the sixth module of the program Full Stack Web Application Development at Canadian Business College.

Site Structure

The primary requirement for this assignment is to create a website using the technologies listed in the next section. The site needs to have 3 pages that can be accessed via the navigation links at the top.

The three pages need to be this home page, a form page, and a page of choice:

  • Home: The home page is an introduction to the site and also includes information about the author of the site.
  • Form: The form page demonstrates receiving a form submitted by a user of this site. The form has several specific fields as well as the ability to upload a photo.
  • Contact: The contact page is the page of choice. It has a standard contact form, allowing a user of this site to send an email to the site author.

Technology Stack

This site is built using modern web technologies. Since this is the first assignment in the module, the stack consists of only a few technologies:

  • Frontend
    • HTML
    • CSS
  • Backend
    • Node.js
    • Express JS Framework

About the Author

The author of this site is a student at Canadian Business College taking the Full Stack Web Application Development diploma program. They have a background in software development, which consists of university education studying Computer Science and multiple online certifications in specific development technologies.

The author's story starts as a teenager where they were first introduced to computer programming. The university education was foundational for the author as it gave them a deep understanding of the science behind programming and software development. This education was pursued around the early 2000s, and as a result, the software development landscape has drastically changed since then.

Most notably, web development did not exist when the author went to university. Web sites were strictly built using frontend technologies like HTML, CSS, and JavaScript. Additionally, web sites were primarily static sites serving up content for consumption and did not emulate a software program that would be installed on a computer.

After about 15 years away from the industry, the author returned to their passion for software development. Due to the massive changes in the industry, the author pursued online certifications via educational platforms such as EdX, Udemy, and Coursera.

The online courses and programs gave the author a good introduction to the current state of development, particularly the fields of web and mobile development. After several years of acquiring and accumulating multiple certificates, the author decided to pursue a local diploma program.
